# 建筑设计师常用网站及抓取策略

本文档供 Agent 在抓取前查阅，根据目标 URL 的域名匹配相应的抓取策略。

## 抓取难度分级

- **低**：WebFetch 可直接抓取，静态 HTML / SSR，无登录墙
- **中**：部分内容可抓，可能有懒加载或局部 SPA，agent-browser 效果更好
- **高**：强制登录、重度 SPA、严格反爬，agent-browser 也未必能完整获取
- **极高**：基本无法通过 Web 端抓取（如强制 App 的平台）

---

## 一、国际建筑/设计平台

### ArchDaily 中文（建日筑闻）
- **域名**：`archdaily.cn`
- **用途**：全球最大建筑媒体的中文版，按建筑类型/材料标签筛选案例，含平面图、剖面图
- **难度**：低
- **策略**：WebFetch 即可。OG 标签完整，文章页结构清晰
- **特殊字段**：关注 `<meta>` 中的建筑师名称、项目地点

### ArchDaily 国际版
- **域名**：`archdaily.com`
- **用途**：同上英文版，内容更全
- **难度**：低～中
- **策略**：WebFetch 可抓多数页面。图片库页面有懒加载，agent-browser 更优

### Dezeen
- **域名**：`dezeen.com`
- **用途**：建筑、室内、产品设计并重，欧洲设计趋势
- **难度**：低
- **策略**：WebFetch 即可。SSR，OG 标签完整

### Designboom（设计邦）
- **域名**：`designboom.com`
- **用途**：建筑、艺术、科技跨界，前沿实验性方案
- **难度**：低
- **策略**：WebFetch 即可。偶有订阅弹窗但不拦截内容

### Divisare
- **域名**：`divisare.com`
- **用途**：高审美建筑摄影合集，被称为"建筑百科全书"
- **难度**：中
- **策略**：项目列表页 WebFetch 可抓。图集为局部 SPA，agent-browser 更优
- **注意**：部分功能需注册

### Landezine
- **域名**：`landezine.com`
- **用途**：全球顶级景观设计媒体，广场/公园/滨水空间
- **难度**：低
- **策略**：WebFetch 即可。WordPress 架构，结构清晰

## 二、国内建筑/设计社区

### 谷德设计网（gooood）
- **域名**：`gooood.cn`
- **用途**：国内流量最大建筑媒体，高质量项目摄影+双语报道。深圳各大事务所发布作品的首选
- **难度**：低
- **策略**：WebFetch 即可。WordPress 架构，无登录限制
- **特殊字段**：文末通常有材料清单和建筑师信息

### 有方（Archiposition）
- **域名**：`archiposition.com`
- **用途**：国内学术气质最强的建筑媒体，深度评论、建筑考察
- **难度**：低
- **策略**：WebFetch 即可。SSR 为主

### 建筑档案（ARCH-ive）
- **域名**：`archivechina.com`
- **用途**：视频采访形式记录建筑师创作历程
- **难度**：中
- **策略**：文字部分 WebFetch 可抓。核心内容是视频，需告知用户视频需手动查看

### 专筑网（iArch）
- **域名**：`iarch.cn`
- **用途**：方案草图、参数化教程、竞赛资讯
- **难度**：中
- **策略**：文章页 WebFetch 可抓。资料下载需注册

### ABBS 建筑论坛
- **域名**：`abbs.com.cn`
- **用途**：国内最早的建筑专业论坛
- **难度**：低
- **策略**：WebFetch 即可。传统论坛架构

### 建筑中国（ARCHINA）
- **域名**：`archina.com`
- **用途**：行业大数据、企业排名、品牌影响力报告
- **难度**：低
- **策略**：WebFetch 即可。静态内容为主

### 景观中国
- **域名**：`landscapecn.com`
- **用途**：景观设计专业媒体，植物与场地材料
- **难度**：低
- **策略**：WebFetch 即可

### 建筑学院（ArchCollege）
- **域名**：`archcollege.com`
- **用途**：案例图片、CAD 素材、SU 模型、软件教程
- **难度**：中
- **策略**：文章 WebFetch 可抓。资源下载需注册

### 筑龙网
- **域名**：`zhulong.com`
- **用途**：施工图纸、节点详图、技术规范下载
- **难度**：高
- **策略**：资料下载强制登录+积分。论坛标题可抓，详细内容需登录
- **建议**：仅抓取文章列表和标题，标注"详细内容需登录查看"

## 三、图片灵感类

### Pinterest
- **域名**：`pinterest.com`
- **用途**：做情绪板（Mood Board），用英文关键词搜索空间氛围参考图
- **难度**：高
- **策略**：强制登录后才能看完整内容，WebFetch 基本无效。agent-browser 可抓登录前首屏
- **注意**：国内需翻墙

### Behance
- **域名**：`behance.net`
- **用途**：设计师作品集，建筑表现图/效果图/手绘质量极高
- **难度**：中～高
- **策略**：无登录可浏览项目。agent-browser 推荐——页面是 SPA 懒加载
- **注意**：WebFetch 大概率只拿到空壳 HTML

### Dribbble
- **域名**：`dribbble.com`
- **用途**：UI/视觉设计灵感
- **难度**：中
- **策略**：列表页部分可 WebFetch，详情页需 agent-browser

## 四、材料/产品类

### MC 材料馆
- **域名**：`mc-materials.com`
- **用途**：建筑材料库，材料样板照片+性能参数+应用案例
- **难度**：中
- **策略**：材料展示页 WebFetch 可抓。部分参数需注册

### BIMobject
- **域名**：`bimobject.com`
- **用途**：BIM 对象平台，国际品牌发布 Revit/IFC 模型
- **难度**：中
- **策略**：产品列表页 WebFetch 可抓，详情页部分为 SPA

## 五、已知极难抓取的站点

以下站点 WebFetch 基本无效，agent-browser 也困难，遇到时应**提前告知用户**：

| 域名 | 原因 |
|------|------|
| `xiaohongshu.com` | 强制 App/登录，Web 端内容极少，反爬极严格 |
| `weixin.qq.com` / `mp.weixin.qq.com` | 微信公众号，动态 URL 会过期 |
| `zhihu.com` | 登录墙 + 反爬 |
| `zhulong.com`（下载资源）| 登录 + 积分墙 |

## 抓取决策快速参考

```
收到 URL → 提取域名 → 查本文档
  ↓
域名匹配到"低难度"站点 → WebFetch 即可
域名匹配到"中难度"站点 → 有 agent-browser 就用，没有就 WebFetch 尽力抓
域名匹配到"高/极高"站点 → 告知用户该站点抓取成功率较低，然后尽力抓取
域名不在列表中 → 按默认流程执行（agent-browser 优先，WebFetch 兜底）
```
